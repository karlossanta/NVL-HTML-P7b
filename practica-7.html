Periféricos de un Ordenador

Índice

A C M R T

Glosario

Altavoces

Es un periférico de salida. Reproduce el audio del ordenador.

Cámara web

Es un periférico de entrada. Es utilizado para enviar imágenes a la computadora, por ejemplo para realizar videollamadas o para grabar las imágenes en el disco duro.

Monitor

Es un periférico de salida. Se utiliza para mostrar información útil al usuario.

Actualmente existen también monitores con pantallas táctiles, en este caso serían periféricos de entrada y salida.

Micrófono

Es un periférico de entrada. Se utiliza para grabar audio, establecer conversaciones o incluso para el reconocimiento de voz.

Ratón

Es un periférico de entrada. Es utilizado para facilitar el uso del entorno gráfico del ordenador.

Teclado

Es un periférico de entrada. Consiste en un conjunto de teclas que al presionarse envían la información de la tecla pulsada al computador.
